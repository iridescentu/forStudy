SELECT * FROM employees;

- 특정 컬럼 읽기
```sql
SELECT 
    employee_id, 
    first_name, 
    last_name, 
    hire_date
FROM
    employees;
```

- 간단한 계산
```sql
SELECT 
    first_name, 
    last_name, 
    salary, 
    salary * 1.05
FROM
    employees;


SELECT 
    first_name, 
    last_name, 
    salary, 
    salary * 1.05 AS new_salary
FROM
    employees;
```
